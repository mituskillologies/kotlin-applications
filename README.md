## üíª How to Run

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/mituskillologies/kotlin-applications.git](https://github.com/mituskillologies/kotlin-applications.git)
    ```
2.  **Open in Android Studio:**
    * Launch Android Studio.
    * Select "Open an Existing Project".
    * Navigate to the cloned directory and select the specific project folder you wish to run (e.g., `01 Login & Calculator`).
3.  **Build and Run:**
    * Wait for Gradle to sync.
    * Select an Emulator or a physical Android device.
    * Click the **Run** (Green Arrow) button.
